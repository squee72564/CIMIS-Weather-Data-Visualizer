CIMIS Weather Data Visualizer

This website serves as a basic visualizer for the CIMIS weather station data. It can be used to select a station and display several data attributes for all daily data collected at the sensor since its installation.

The data has been manually collected from the CIMIS website and will be updated periodically. All data is untreated and as is from the sensor data, including outliers, sensor malfunctions, and any other instances that would cause funky data.

Click these links to find out more about the weather sensor placement, sensor specifications, and sensor maintinenance here.

The website uses Flask, matplotlib, scikit-learn, and other python libraries to render the web layout and generate plots for the data.
